# VitePress 个人网站重构实录：从零构建极简数字花园

搭建一个属于自己的个人网站一直是我的一个愿望。经历了数次迭代，我最终选择了 **VitePress** 作为我的主力建站工具，并在这个周末完成了一次全面的架构与视觉重构。

## 为什么选择 VitePress？

在此之前，我尝试过 Hexo、Hugo 等传统的静态生成器，但作为 Vue 的忠实拥趸，VitePress 给我的体验是无可比拟的：
1. **极速的热更新**：基于 Vite 构建，无论项目多大，保存即刻在浏览器呈现。
2. **Markdown 混合 Vue**：可以直接在 Markdown 中使用自定义的 Vue 组件，这极大地扩展了页面的表现力。
3. **现代化的默认主题**：自带美观的默认样式、本地搜索和响应式布局，省去了大量基础配置工作。

## 视觉重构：极简与高级感的平衡

在本次重构中，我主要参考了 [ReactBits](https://reactbits.dev/) 和 [rauno.me](https://rauno.me/) 的设计美学，确立了**“深色优先、微动画点缀、双字体系统”**的设计基调。

### 1. 双字体系统
我引入了类似 Claude 文档的排版思路：
- **正文与标题**：使用开源衬线体 **Lora**。衬线体具有极强的书卷气息，让大段的文字阅读起来更具“沉浸式阅读”的质感。
- **UI 控件与导航**：保留无衬线体 **Inter**。在按钮、侧边栏等需要高效率识别的组件中，无衬线体更加清晰现代。

### 2. 微动画与光效
为了打破静态页面的沉闷感，我通过 CSS 自定义了许多微妙的交互细节：
- 首页的卡片（Feature / Project）悬浮时，不仅有轻微的 Z 轴上浮，边框还会呈现出与品牌色（Indigo/Violet）一致的霓虹发光效果。
- 导航栏采用了 `backdrop-filter: blur(12px)` 的磨砂玻璃效果，随着页面的滚动，下方的文字若隐若现。

## 架构的梳理与组件化尝试

利用 VitePress 的 `enhanceApp`，我封装了 `<ProjectCard>`、`<Timeline>` 和 `<SkillBadge>` 三个 Vue 组件，并在 `.md` 文件中直接调用。这让我的“关于”和“项目”页面不仅排版精美，而且后续维护成本极低，添加新经历只需加一行组件标签即可。

## 结语

这个小小的“数字花园”不仅仅是我对外展示的简历，更是我在技术探索和跨学科学习中的一个锚点。未来我将在这里记录更多的学习笔记和项目复盘，见证自己的每一步成长。
