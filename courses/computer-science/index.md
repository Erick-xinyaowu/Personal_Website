# 计算机科学课程资料

这里汇总了计算机科学与技术方向的课程资料。

## 核心课程

- **数据结构** — 基本数据结构与算法分析
- **计算机网络** — 网络通信协议与网络应用
- **计算机组成与体系结构** — 计算机硬件系统的基本组成和工作原理
- **离散数学** — 集合论、图论、组合数学等
- **软件工程** — 软件开发流程与项目管理
- **Java 应用开发** — Java 语言及面向对象编程
- **程序设计 A** — 编程基础与进阶
- **数字逻辑与数字电路** — 数字逻辑设计与电路

## 编程语言

- **Python 语言** — Python 基础与应用
- **大学计算机** — 计算机基础概论
