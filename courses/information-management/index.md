# 信息管理课程资料

这里汇总了信息管理与信息系统专业方向的课程资料。

## 核心课程

- **信息管理学基础** — 信息管理的基本概念、原理与方法
- **信息组织** — 信息资源的描述、标引、分类与编目
- **信息检索** — 信息检索原理、技术与系统
- **信息分析** — 信息分析方法与实践
- **信息可视化** — 信息可视化理论与技术
- **数据可视化** — 数据可视化设计与实现
- **数据挖掘** — 数据挖掘算法与应用
- **数据库原理** — 数据库设计与 SQL 编程
- **信息系统分析与设计** — 信息系统的需求分析、设计与实现
- **信息服务与用户** — 信息服务理论与用户研究
- **人机交互** — 人机交互设计原理与方法
- **社会网络计算** — 社会网络分析理论与计算方法
- **社会调查与统计分析** — 社会调查方法与统计分析技术
